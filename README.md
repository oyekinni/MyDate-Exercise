# MyDate - Exercise 10.14

A Java class that represents a date using year, month (0-based), and day.
Supports construction from elapsed milliseconds since the Unix epoch (Jan 1, 1970).

## Files
- `MyDate.java` — the MyDate class implementation
- `TestMyDate.java` — test program

## How to compile and run
javac MyDate.java TestMyDate.java
java TestMyDate

## Sample Output
Date 1 (current date):
  Year:  2026
  Month: 2
  Day:   30

Date 2 (34355555133101 ms since epoch):
  Year:  3058
  Month: 10
  Day:   4
